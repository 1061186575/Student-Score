# 学生成绩管理系统


### 启动方式
先把mysql.js的"数据库连接"配置成自己的数据库,
然后把自己的数据库里的字段改成mysql.js里对应的字段,
并启动自己的数据库,然后执行下面的操作.

``` bash
# 1.安装node.js (下载安装即可)

# 2.安装依赖
npm install express mysql --save-dev

# 3.运行服务 
npm run dev

```
然后打开浏览器访问: <http://localhost:3000/>
